# App web Expensive tracker ( Caixa para controle financeiro )

Uma aplicação onde o usuário tem sua carteira de dinheiro e um painel onde pode adicionar seus lucros e gastos de forma intuitiva. 
A aplicação realiza o cálculo e mostra para o usuário o resultado de forma automática, de acordo que ele vai adicionando os items e seus valores no painel.
